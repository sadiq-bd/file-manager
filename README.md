# Simple File Manager

Simple File Manager for managing files on the server

## Requirements
```bash
php >= 8.0
```

## Installation

Clone this repo
```bash
git clone https://github.com/sadiq-bd/file-manager.git
```
Install dependencies 
```bash
composer update
```
Edit .env as needed 

That's all
